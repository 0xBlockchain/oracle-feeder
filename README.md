# oracle-feeder
Oracle Feeder Daemon

## Usage
terrafeeder --from {account} --pass {password} 

### Global Flags

**--home** :
set target directory to save history data

**--no-rest** : 
run without REST API serving

**--no-voting** : 
rin without Voting prices

**--laddr** : 
set listening address for REST APIs

### Updater Flags

**--updating-source** :
